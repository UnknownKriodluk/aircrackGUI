# EN/Английский

## aircrack-ng GUI
GUI for AirCrack-ng
**ONLY FOR LINUX**

Automatically saves the handshake to the specified file

Supports linux distributions with these package managers: apt, dnf, yum, pacman

**Tested on Manjaro 22.1.1 + Python 3.10.10**

## Launch
python aircrackGUI_(version).py
